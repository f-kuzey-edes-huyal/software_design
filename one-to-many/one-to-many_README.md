# Teacher-Student One-to-Many Example in Java

## Overview
This project demonstrates the **one-to-many relationship** in Java using **Object-Oriented Programming (OOP)** principles.  

- A **Teacher** can have multiple **Students**.  
- Shows **composition** and basic **OOP concepts** such as classes and lists.

---

## How to Compile and Run

1. **Open a terminal or PowerShell** in the project folder containing the `.java` files.  

2. **Delete old compiled class files** (optional but recommended):

```del *.class```

3. Compile all Java files for Java 8 compatibility:
```javac --release 8 *.java```

```java Main````
