# Starbuzz Coffee - Decorator Pattern Example

## Files included
- `Beverage.java`
- `CondimentDecorator.java`
- `Espresso.java`
- `HouseBlend.java`
- `DarkRoast.java`
- `Decaf.java`
- `Mocha.java`
- `Soy.java`
- `Whip.java`
- `SteamedMilk.java`
- `StarbuzzCoffee.java`

## How to compile and run

1. Open a terminal in this folder.
2. Compile the files:

   ```javac *.java```
   

3. Run
   ```java StarbuzzCoffee```
